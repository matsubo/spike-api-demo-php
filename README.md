SPIKE Checkout sample code
==============================


Live demo
------------------

- See
  - http://matsu.teraren.com/static/spike_checkout/

Requirements
------------------
- bower
- PHP 5.3=<
- composer


Setup
-------------------

```
% git clone git@github.com:matsubo/spike-checkout.git
% cd spike-checkout
% bower install
% php composer.phar install
% echo 'YOUR SECRET KEY' > secret.key
```
